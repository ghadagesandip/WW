POWER

The data center provides redundant power systems with an Inline power backup using World Class Eaton Powerwave and GE UPS systems in N+1 mode with 15 mins battery backup and dedicated diesel generators.

The UPS, controls constantly monitors the quality of power so that all equipment receives 240-volt power, all the time. Moreover, the UPS and generators are tested weekly to avoid power disruptions.

Salient Features

  • Critical IT Power system with N+ 1 redundancy
  • Dual UPS power source at each rack powered through ATS
  • 32 AMP Breakers for each rack.
